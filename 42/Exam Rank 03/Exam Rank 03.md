# Exam Rank 03

> Il y a 2 niveaux : **la lecture** et **le backtracking**

---

## Niveau 1 — La lecture

| Exercice | Concept         | fd         |
| -------- | --------------- | ---------- |
| filter   | stdin + pattern | 0 -> 1      |
| gnl      | lecture ligne   | fd fichier |

Le niveau 1 est constitué de 2 exercices : [[42/Exam Rank 03/Level 1/filter\|filter]] et [[42/Exam Rank 03/Level 1/broken_GNL\|broken_GNL]]. Ils sont basés sur la lecture avec [[Langages/C/read\|read]] et [[Langages/C/open\|open]], via les [[Langages/C/File descriptors\|file descriptors]], soit depuis la ligne de commande, soit directement depuis un fichier.

### GNL
- Buffer statique
- Retourne une ligne à la fois

## Niveau 2 — Le backtracking

| Exercice     | Principe                |
| ------------ | ----------------------- |
| permutations | Toutes les combinaisons |
| n_queens     | Placer N reines         |
| rip          | Équilibrer parenthèses  |

Le niveau 2 est constitué de 3 exercices : [[42/Exam Rank 03/Level 2/permutations\|permutations]], [[42/Exam Rank 03/Level 2/rip\|rip]] et [[42/Exam Rank 03/Level 2/n_queens\|n_queens]]. Ils sont basés sur le [[backtracking]] a fin de testé toutes les possibilités et revenir en arrière quand on est bloqué.
